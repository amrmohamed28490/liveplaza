Odoo Marketplace Pre-Order

Make even out of stock a selling opportunity

The module allows you to take and manage pre-orders on your Odoo marketplace website. The customer can preorder the out-of-stock products so you do not lose any potential revenue due to product unavailability. You can set pre-order settings for products at both globally or product level.

Detailed Features List

Take Pre-orders on Odoo Marketplace Website

  • The module offers the features to sell products as pre-orders on Marketplace.
  • The pre-orders can be taken for out-of-stock products or in stock products.
  • The admin and seller both can also add a pre-order feature to each product in Odoo.

Configure Pre-order settings at Product level in Odoo

  • The admin can set pre-order configuration at global level.
  • Set pre-order quantities, payment options, products page message etc. using global settings
  • The admin and seller can also change the pre-order settings at individual product level.

Pre-order notification on shop page

  • Automatically adds the ribbon message ‘Available for Pre-order’ on respective products.
  • The ribbon notification is available for grid and list view on the Marketplace shop page.

Add details for pre-order on Odoo Marketplace Product page

  • The admin can set detailed message details for pre-order on the product page.
  • You can also add a message regarding payment criteria for placing pre-order. 
  • You can also show the availability date for the pre-order product.

Set Payment options to place Pre-orders on Odoo Marketplace

  • You can set the complete or partial payment option for customer placing pre-orders.
  • The setting can be managed at global level or each product level.

Set threshold product quantities needed to place pre-order

  • Set minimum and maximum quantities that can be pre-ordered for a product.
  • The settings can be managed at global and product level.

Show pre-order deadline for customers on product page

  • You can choose to show the last date for placing the pre-order on the respective product page.

Send mail to customer after availability For payment completion

  • Configure automatic mail to be sent to customers once the product is in stock.
  • The mail is sent to customers who made partial payment during pre-order.
  • Customize the template to ask for making the complete payment so order can be shipped.
