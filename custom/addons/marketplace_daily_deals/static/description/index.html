Odoo Marketplace Daily Deals And Flash Sales

Basic Features

  • Marketplace Daily Deals is used to create offers/deals in the Marketplace.
  • Sellers in a Marketplace can create their own deal.
  • Sellers can add only their own products in the deal.
  • Admin can create a deal of any of the seller's product related to the marketplace.
  • Seller can set pricelist when creating new Daily deal/ offer.
  • Seller can also Customize website view as per his need.
  • Seller can manage Message and Notification settings.
  • It gives a separate menu for creating only deals and offers.
  • Only validated deals are displayed on the website.
  • A deal will have a start date and expiry date.
  • Displayed on the website in a separate menu 'Deals and Offers'.

Marketplace Daily Deals

Website View

View of daily deals in website.

Seller Side View

Seller can view all their own deals.

Seller can enter the details of the deal/offer.

Seller can also customize website view for each deal.

Seller can add their own products after the deal is validated.

Admin Side View

Admin can view the deals of all the sellers of marketplace.

Admin can create and enter the details of the deal/offer.

Admin can add the products after the deal is validated.
